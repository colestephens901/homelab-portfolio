Any screenshots added here will have sensitive details removed or blurred, including:

* Public IP addresses
* Private IP addresses
* Internal hostnames
* Domain names if needed
* Usernames
* API tokens
* Credentials
* Exact firewall rules
* Management URLs
* Device serial numbers
* Any other identifying or sensitive information
